Optimus

Web Framework

[Architect] [Lead] [UX] [Dev]
[C#] [ASP.net MVC] [Razor] [AutoMapper] [jQuery] [Bootstrap] [DataTables.net] [HTML] [less] [NuGet]
Conceptualized a web framework after noticing the repetitive nature of rich web applications and the need for:

  • Sharing common libraries and their accompanying adaptations such as AutoMapper, jQuery and DataTables.net
  • Encouraging web development & UX best practices (e.g. XSRF Tokens, Post/Redirect/Get) through familiar code patterns such as inheritance & interface implementation
  • Simplifying the update & use of underlying CSS & JavaScript resources

Main Contributions:

  • Implemented initial version including runtime scaffolding in Razor views, JavaScript Behaviors
  • Sold the team on the value & potential
  • Lead system design & worked with several team members to implement framework alongside product releases
  • Reached out for user & developer feedback to improve the framework
  • Mentored contributing developers on growing the framework and helped them achieve ownership of the code

Impact:

  • Enabled developers with little web experience to develop sophisticated pages in hours or days instead of weeks
  • Significantly improved the user experience across our systems for both our customers & operations team
  • Eased transition to MVC architecture which greatly improved maintainability of code
